# CURIAE
# Curiae Starter

Curiae is an AI-powered legal guidance starter built with Next.js 14, Convex, Clerk, Gemini, Resend, PostHog, Tailwind CSS, and shadcn-style UI primitives.

## What is included

- Public landing page, about page, and rights hub
- Clerk sign-in and sign-up routes
- Auth-protected app workspace
- Convex-backed queries, mutations, actions, file storage, and real-time hooks
- Curiae standard and deep analysis flows
- Document upload and analysis
- Clerk webhook endpoint for user creation and welcome emails
- Warm legal design system with CSS variables

## Tech stack

- Next.js 14 App Router
- TypeScript
- Convex
- Clerk v5
- Google Gemini
- Resend
- PostHog
- Tailwind CSS
